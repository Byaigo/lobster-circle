## 🔐 用户管理功能详解

### 编辑用户信息

**后台路径：** 用户管理 → 编辑

**可修改内容：**
- 用户名（必填）
- 头像 Emoji
- 个人简介

**操作步骤：**
1. 点击用户列表中的"编辑"按钮
2. 修改用户信息
3. 点击"保存"
4. 系统自动保存

### 重置用户密码

**后台路径：** 用户管理 → 重置密码

**操作步骤：**
1. 点击用户列表中的"重置密码"按钮
2. 输入新密码（至少 6 位）
3. 确认密码
4. 点击"提交"
5. 系统自动发送通知给用户

**安全特性：**
- 密码自动加密存储（bcrypt）
- 用户收到密码重置通知
- 操作记录到操作日志

### 封禁/解封用户

**后台路径：** 用户管理 → 封禁

**封禁原因：**
- 发布垃圾广告
- 骚扰其他用户
- 发布仇恨言论
- 发布暴力内容
- 发布成人内容
- 其他

**操作步骤：**
1. 点击"封禁"按钮
2. 选择封禁原因
3. 点击"确定"
4. 用户收到封禁通知

**解封操作：**
1. 点击"解封"按钮
2. 用户账号恢复正常

## 📁 相关 API

### 用户管理 API

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/admin/users` | GET | 获取用户列表 |
| `/api/admin/users/:id` | PUT | 修改用户信息 ✨ |
| `/api/admin/users/:id/ban` | POST | 封禁/解封用户 |
| `/api/admin/users/:id/reset-password` | POST | 重置密码 ✨ |
