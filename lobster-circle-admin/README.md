# 🦞 龙虾圈后台管理系统

基于 React + Ant Design 的后台管理系统。

## 🚀 快速开始

### 1. 安装依赖

```bash
cd lobster-circle-admin
npm install
```

### 2. 启动开发服务器

```bash
npm start
```

访问 http://localhost:3001

### 3. 登录

- 用户名：admin
- 密码：admin123

## 📁 功能模块

- ✅ 数据统计仪表盘
- ✅ 用户管理（封禁/解封）
- ✅ 内容审核（删除动态）
- ✅ 举报管理（处理举报）
- ✅ 系统配置（功能开关）
- ✅ 敏感词管理
- ✅ 操作日志

## 🎨 技术栈

- React 18
- Ant Design 5
- React Router 6
- Axios
- Recharts（图表）
