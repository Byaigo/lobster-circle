## 🔧 需要优化的功能

### 性能优化

#### 1. 数据库索引优化 ⚠️
**现状**: 部分查询缺少索引  
**建议**:
```javascript
// User 模型
userSchema.index({ username: 1 });
userSchema.index({ createdAt: -1 });

// Post 模型
postSchema.index({ userId: 1, createdAt: -1 });
postSchema.index({ hashtags: 1 });

// Message 模型 (已有)
messageSchema.index({ conversationId: 1, createdAt: -1 });
```

#### 2. API 响应缓存 ⚠️
**现状**: Redis 中间件已创建但未使用  
**建议**:
```javascript
// 信息流缓存 5 分钟
app.use('/api/posts/feed', cache(300));

// 用户信息缓存 10 分钟
app.use('/api/users/:id', cache(600));
```

#### 3. 图片压缩 ⚠️
**现状**: 图片直接上传，未压缩  
**建议**:
- 前端：上传前压缩
- 后端：sharp 库处理
- CDN：自动压缩

#### 4. 请求限流 ⚠️
**现状**: 无限流保护  
**建议**:
```javascript
const rateLimit = require('express-rate-limit');

const limiter = rateLimit({
  windowMs: 15 * 60 * 1000, // 15 分钟
  max: 100 // 最多 100 次请求
});

app.use('/api/', limiter);
```

### 安全优化

#### 5. XSS 防护 ⚠️
**现状**: 用户输入未过滤  
**建议**:
```javascript
const xss = require('xss-clean');
app.use(xss());
```

#### 6. SQL 注入防护 ⚠️
**现状**: 使用 Mongoose（已防护）  
**建议**: 保持现状，注意原生查询

#### 7. 敏感信息脱敏 ⚠️
**现状**: 日志可能输出敏感信息  
**建议**:
```javascript
// 密码不输出
userSchema.methods.toJSON = function() {
  const obj = this.toObject();
  delete obj.password;
  return obj;
};
```

#### 8. 登录失败限制 ⚠️
**现状**: 无限制  
**建议**:
```javascript
// 5 次失败后锁定 15 分钟
const failedAttempts = await LoginLog.countDocuments({
  userId,
  status: 'failed',
  loginAt: { $gte: new Date(Date.now() - 15*60*1000) }
});

if (failedAttempts >= 5) {
  return res.status(429).json({ error: '尝试次数过多' });
}
```

### 用户体验优化

#### 9. 加载状态 ⚠️
**现状**: 部分页面缺少加载指示器  
**建议**: 统一使用 ActivityIndicator

#### 10. 错误提示 ⚠️
**现状**: 错误提示不统一  
**建议**: 统一错误处理组件

#### 11. 空状态 ⚠️
